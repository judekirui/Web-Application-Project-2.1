# Web-Application-Project-2.1

A multi-page HTML website for a construction and heavy-lifting company that leases out units. It includes 9 pages covering services, equipment, past projects, safety info and a contact/booking form.

Below are the pages to be created:

style.css - Default page design
nav.html - Interlinking the various pages
index.html – Home page
about.html – Company history
services.html – Services offered
fleet.html – Equipment catalog
projects.html – Past project portfolio
contact.html – Contact form and quote requests
faq.html

Built With:
HTML5
CSS3
JavaScript
Bootstrap
